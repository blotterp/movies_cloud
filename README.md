# Веб-приложение для просмотра фильмов.
# На текущий момент реализована только небольшая часть дизайна.

## Использованный стек: HTML+CSS+JavaScript
## Интерфейс: 
### 1. На странице выведен каталог карточек фильмов. Каждая карточка содержит изображение, заголовок и краткое описание. 
### 2. Карточки располагаются в виде сетки и удобно просматриваются на разных устройствах: на компьютере — 6 карточек в ряд, на мобильных — по одной.
